# Projeto-Final-Descomplica-VNT

# Lista de Compras e Remédios

Este é um aplicativo pequeno e rápido para ajudar você a criar e organizar listas. Você pode usar ele para fazer listas de compras ou listas de remédios.

Tudo funciona direto no seu navegador, sem precisar de internet ou programas extras.

## Como Funciona?

O projeto usa apenas o básico da web:

HTML: Para montar a página, os botões e exibir a tabela criada dinâmnica.

CSS: Para deixar a página com uma boa aparência (cores, tamanhos).

JavaScript (JS): É o "cérebro" que faz as coisas acontecerem (salvar seu nome, adicionar itens na lista, editar e excluir).

## Funcionalidades

- Boas-vindas Personalizada: Você coloca seu nome, e o app te cumprimenta.

- Escolha o Tema: Você pode escolher se quer fazer uma lista de Compras ou de Remédios.

- Adicionar: Coloque o nome e a quantidade de qualquer item.

- Organizar: Todos os itens aparecem em uma tabela limpa.

- Editar e Excluir: Você pode mudar o nome ou a quantidade de um item, ou tirá-lo da lista a qualquer momento.

## Como rodar o projeto?

Salve os arquivos (index.html, script.js e index.css) no mesmo lugar no seu computador.

Clique duas vezes no arquivo index.html.

O aplicativo abrirá no seu navegador (Chrome, Firefox, etc.).
